Atm Machine & Mobile App

ROLE: UX - UI

INSTRUCTOR: MARCO CATANI

PROJECT BRIEF

ATM has several interfaces the public interacts to complete their daily commuting tasks which includes a touch screen kiosk and mobile app. In this project, the goal is to evaluate these user interfaces, identify the weaknesses and how to improve the overall user experience. How the public interacts with these interfaces, what types of main tasks the public completes, and understanding the user journey will define how these interfaces can improve to better the immediate usability.

QUESTIONS

On how the public interacts with these interfaces, what types of main tasks the public completes, & understanding the current user journey.

METHODOLOGY

Observations, Interviews, Personas, Card Sorting, Prototyping, User Testing.
